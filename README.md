# 🐸ScratchPaper Project (2022/9/22)
낙서장 게시판 대작전
## USE
- Python
- Flask
- JWT
- jinja2
- JS
## API
- AUTH
> JWT 기반 회원가입 + 로그인 + 로그아웃 
- POST
> 게시물 전체조회 + 상세페이지 + 작성(캔버스) + 좋아요
>> 펜 브러쉬 크기, 색상 조정 가능
>>> 이미지 다운로드 (BLOB)
